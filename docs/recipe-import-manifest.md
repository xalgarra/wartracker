# Manifiesto de importación de recetas

Fuente local: `recetas/`

Objetivo: preparar las fotos actuales para cargarlas como `recipes`, `recipe_photos` y `recipe_paints`.
Los pasos se preparan para `recipe_steps` con `technique` e `instruction`.

Notas:
- Las fotos tipo captura de WhatsApp incluyen imagen + instrucciones; conviene guardarlas como fotos de referencia de la receta.
- Los nombres de pinturas están transcritos visualmente. Las entradas marcadas como `revisar` necesitan confirmación antes de importar.
- Las fotos JPEG finales son originales sin texto y pueden usarse como foto limpia de portada o como duplicado de apoyo.
- No se modelan mezclas. Si una receta usa una mezcla puntual, queda escrita en `instruction`.
